# YTERMUX
YTERMUX ejecuta de forma legible la extracción streaming de YT-DLP.

<h2>EJECUCIÓN</h2>
<p>~ $ yes | pkg install python</p>
<p>~ $ pip install yt-dlp</p>
<p>~ $ yes | pkg install ffmpeg</p>
<p>~ $ cd YTERMUX</p>
<p>~/YTERMUX $ chmod +x YTERMUX.py
<p>~/YTERMUX $ mv YTERMUX.py </p>
<p>~ $</p>
